# @codloy/utils

Utility functions for every project

## Installation

```bash
npm i @codloy/utils
```

## Usage

```typescript
import { isNumber } from '@codloy/utils';
// or
import isNumber from '@codloy/utils/isNumber';

console.log(isNumber(123)); // true
console.log(isNumber('123')); // false
```
